surfit short description.


surfit provides the algorithms for high-detail maps calculation by source data of different extent of truth. Then making a map, it is necessary to apply a universal method of usage of different data - from precise to inconsistent. Such a problem of usage of any more or less reliable information arises when it is insufficient, which is actual in oil-fields researches.
We provide our realisation of the theory of step-by-step minimization of functional sequences. This theory takes into account information of different extent of truth about object of interest. surfit is a computer-programs for gridding, able to work with great amount of geological, geophysical, expert and other kinds of information.

What surfit can do:

  • save maps to Surfer ASCII format
  • save maps to binary ROFF (Roxar open file format). This format uses as surfit-internal format.
  • surfit uses Tcl as internal script language. So, if you want to calc the map, you must to write a script-programm (see examples).

What surfit can not do:

  • maps printing.
  • programm for map calculation with user-friendly interface.

surfit consists of:

  • libsurfit.dll

    library with 1D and 2D algorithms:

    • algorithms for maps calculation by source data of different extent of truth (method of step-by-step minimization of functional sequences)
    • wavelet-analysis algorithms (decomposition and reconstruction), applicable to maps.
    • Systems of linear algebraic equations solvers:
      • conjugate gradients (cg)
      • simple iterations (Richardson)
      • special multi-grid iterative method
    • isolines tracing algorithm (conrec).

  • libsurfit_gl.dll

    Visualisation library. Uses for display of 1d and 2d functions (maps) with isolines.

  • bula.exe

    surfit graphical console.

  • funner1d.exe

    Programm with user-friendly :) interface. Provides easy visualisation of 1d-functions with different detalisation (uses wavelet-analysis).

  • funner2d.exe

    Programm with user-friendly :) interface. Provides easy visualisation of 2d-functions (maps) with different detalisation (uses wavelet-analysis).

  • tclsh84.exe, tcl84.dll Tcl - scripting language. libsurfit and libsurfit_gl extend tcl with additional commands.
