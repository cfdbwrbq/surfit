Introduction

surfit is a computer program which enables to recalculate scattered data to regular grid (gridding). surfit implements the original gridding method CMOFS, which can deal with various data of different extent of truth. surfit is an Open Source and Free Software, the source code is available free of charge under the Free Software GPL license.

The code is written in C++ with STL usage. Careful attention is paid to performance related issues. The initial goal of surfit is to provide an efficient tool to scientists dealing with 3D surfaces computational tasks.

The program has the following basic features:

  • Basic input is a simple ASCII file containing  (X,Y,Z)  coordinates of data-points irregularly spaced in some region.
  • The surface can be defined not only by points, but also by:
    • trend surfaces, which can define surface behaviour in areas of low density data.
    • so called fault lines, which can define discontinuity in the generated surface.
    • isolines and contours, which can define surface values on curves.
  • It is possible to set for each data-object (point, curve, function) how the generated surface should pass through them (exactly or approximative).
  • Smoothness of resulting surface can be controlled by one parameter, like in Splines with Tension method.
  • Can deal with HUGE data sets.
  • Huge grid can be displayed with help of wavelet analysis.
  • Quality of resulting surface is comparable with a surface created by Kriging or Minimum curvature method.
