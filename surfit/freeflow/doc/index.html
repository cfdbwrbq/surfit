Introduction

Energy state of a productive oil formation is an important indicator used for the correct and effective planning of its further development. freeflow is a computer program which implements the way of pressure fields reconstruction based on the constructing and consecutive minimization of functional sequence (CMOFS algorithm).  CMOFS   can deal with various data of different extent of truth. freeflow uses mathematical model describing fluid filtration using the langauge of functionals. freeflow is an Open Source and Free Software, the source code is available free of charge under the Free Software GPL license.

The code is written in C++ with STL usage. Careful attention is paid to performance related issues.

The program has the following basic features:

  • Basic input is a simple ASCII file containing  (X,Y,Z)  coordinates of data-points (wells) irregularly spaced in some region. Using wells we can set for each well downhole pressure or debit/rate, or set them both.
  • The pressure field also be defined not only by points, but with:
    • tectonic zones
    • boundaries of non-productive formations (undefined pressure values)
    • horizontal (directional) wells and formation fractures, which can define both pressures and debits/rates.
  • Can deal with HUGE data sets.
  • Huge grid can be displayed with help of wavelet analysis.
